Сборка исходников осуществляется командой make
Если имя вашего компилятора отличается от "g++", то нужно подкорректировать переменную CXX

Инструкция:
1) Сначала запускаем сервер
2) Затем запускаем скрипт

Разработку делал на raspberrypi 3 (raspbian), в Qtcreator. Использовал только Stl.

Тестировал на Rasbian, lubuntu, работает нормально. На Cygwin как то не очень.
